#How to use the program:


---------------------------------------------------------
HOW TO USE:
---------------------------------------------------------
inside the directory folder go to src folder and launch the program from here: 

cd src

javac *.java

java TranslatorWindow


---------------------------------------------------------
GUI MEANING
---------------------------------------------------------
1. Left side text area is for text input

2. Right side text area is for text output

3. Button "Translate" is for translation. Press the button after the text in the left side text area has been input

4. Button "English - Spanish" shows which way the translation is supposed to happen. Press the button to change translation way.

5. If the text area is not sufficient enough for the input or output text, a scroll bar will appear near each text area

6. To exit the translator press exit button in the top right corner.
